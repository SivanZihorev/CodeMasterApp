# Android Application

Developed in Android Studio, designed for learning to code. Includes connecting to Firebase, YouTube API, and Login/Logout functionality.

## FireBase:
- Real-time Database
- Authentication
- Cloud Firestore

## API:
- YouTube Data API v3
